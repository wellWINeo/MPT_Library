# Library

## Configuration
Config stores in `appsettings.json` file, example below:
```
{
  "ConnectionStrings": {
    "Server": "server=localhost;user=<username>;password=<password>;database=library;"
  },
  "EmailSettings": {
    "email": "user@example.com",
    "name": "Библиотека",
    "password": "<пароль>",
    "host": "<address>",
    "port": <port>
  }
}
```
